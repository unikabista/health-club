# Monroe Health Hub

A community health resource hub for Monroe, Louisiana residents, built with React, JavaScript, HTML, and CSS.

## Features

- **Local Clinics**: Searchable directory of healthcare providers in Monroe
- **Emergency Help**: Critical hotlines and emergency contact information
- **Wellness Resources**: Tips for mental health, nutrition, and healthy living
- **Responsive Design**: Works on all devices

## Technologies Used

- React 18
- React Router DOM
- JavaScript (ES6+)
- HTML5
- CSS3 with Tailwind CSS
- Lucide React Icons

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/unikabista/v0-new-conversation.git
cd v0-new-conversation
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm eject` - Ejects from Create React App (one-way operation)
